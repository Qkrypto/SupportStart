About this demo

SupportStart is a personal portfolio prototype built for learning and demonstration. It is not an official system for any school or district, and it does not submit real tickets. Please don't enter real student or district data, passwords, or other private information. The IT Staff view is intentionally viewable for the demo (access code admin123).

Fewer bad tickets. Faster fixes.

An issue-first assistant that solves the easy things itself and makes the hard things easy to hand off.

Guided, one step at a time

It restates the problem, asks one smart question, then walks through safe fixes with a clear "what to do" and "what you'll see."

Ticket-ready summaries

If a technician is needed, it writes a clean summary: the issue, what was tried, device, and location, ready to paste into any ticket system.

Safe by design

It refuses requests to bypass filters, disable monitoring, or access other accounts, and reminds users never to enter private data.

IT staff console

A separate demo view shows how technicians would review summaries, routing suggestions, failed steps, and improvement metrics.

How it works

Issue-first, not form-first. Contact details are asked only if a technician is actually needed.

Describe the problem

Type it in plain words: "my screen won't turn on," "printer jammed," "can't sign in."

Try a safe fix

The assistant matches your issue and guides you through two to four steps you can do right now.

Get a summary if needed

Still stuck? It collects only what IT needs and writes a clear summary you can copy or download.

Built for a school environment

Safe, private, and welcoming for teachers, students, parents, and staff.

SupportStart will not help anyone:

  • Bypass content filters or web blocks
  • Disable monitoring or device management
  • Access another person's account
  • Get around school or district policy

And it reminds users never to enter passwords, student IDs, grades, medical, or other private information. Legitimate blocked-resource requests are turned into a review request for the right team.
